### What is a `print` statement?

The `print()` function is used to display output to the screen. It helps you see results, messages, and values while your program is running.

### How to use `print` with text and numbers

You can print:

* Text (strings)
* Numbers
* Variables
* A combination of them in one line

### String formatting for dynamic messages

String formatting allows you to create dynamic messages by inserting variables inside text.
Common approaches include:

* Using commas in `print`
* String concatenation
* Formatted strings (f-strings)

This makes your output clearer and more user-friendly.

### Using user input with `print`

You can combine `input()` with `print()` to:

* Ask the user for information
* Display personalized messages
* Create interactive programs
